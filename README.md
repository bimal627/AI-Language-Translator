# AI Language Translator

## 📌 Description
An AI-powered Language Translator developed using Python and Streamlit as part of the CodeAlpha Internship Program. This application translates text between multiple languages through a simple and interactive web interface.

## ✨ Features
- Translate text into multiple languages
- User-friendly interface
- Fast and accurate translation
- Text-to-Speech (TTS) support
- Built with Streamlit

## 🛠️ Technologies Used
- Python
- Streamlit
- deep-translator
- gTTS (Google Text-to-Speech)
- VS Code
- Git
- GitHub

## 📂 Project Structure
```
AI-Language-Translator/
│── app.py
│── requirements.txt
│── README.md
│── translation.mp3
```

## ▶️ Installation

1. Clone the repository

```
git clone https://github.com/bimal827/AI-Language-Translator.git
```

2. Install the required libraries

```
pip install -r requirements.txt
```

3. Run the application

```
streamlit run app.py
```
